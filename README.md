# Physical-Test-Calculator
干净的大学生体测成绩计算器，纯前端实现，无广告无跟踪器。

## 使用方法
- **（推荐）** 通过 [https://lc6464.github.io/Physical-Test-Calculator/](https://lc6464.github.io/Physical-Test-Calculator/) 访问网页版。
- 在 [lc6464/Physical-Test-Calculator-Android](https://github.com/lc6464/Physical-Test-Calculator-Android) 项目中获取 Android 离线使用版本。

## 代码来源
本项目基于 [Polaris-cn/physical-test-calculator](https://github.com/Polaris-cn/physical-test-calculator)，由 Claude 4 开发。<br/>
